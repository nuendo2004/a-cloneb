A CloneB is an Airbnb clone utilized newest technology such as Next.js and Typescript

Live site (https://a-cloneb-git-main-nuendo2004.vercel.app/)

features:
  • Oauth login
  • Post listing
  • Make reservation
  • Third party secured payment 
  • Location service
  • Email message system
